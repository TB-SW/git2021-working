pwd(print workind directory)
: 현재 작업 디렉토리 출력
	/: 루트 (최상단) 디렉토리
	/home : 사용자의 공간
	/home/ubuntu : ubuntu라는 사용자의 디렉터리

mkdir(make directory)
: 디렉토리(폴더) 생성
: mkdir 디렉토리명

ls(list)
: 디렉토리의 파일(디렉터리 포함) 목록을 표시
: ls -al: -a(all), -l(long)
전체파일 목록을 상세하게 표시(list all long)
목록에서 가장 앞에 d표시되어있는 파일 directory
-> 모든 파일 및 상세보기
: . -> 현재 디렉토리
: .. -> 부모 디렉토리
:cd 이동할 디렉터리명

cd .. -> 부모 디렉터리로 이동
cd / -> 루트 디렉터리로 이동
cd /usr -> 루트 디렉터리부터 usr 디렉터리로 이동 
	-> 절대경로(absolute path) 표기법
	:루트 경로부터 도착지까지 적는 방법
cd ~ 또는 cd: -> 홈디렉터리로 이동
cd ../../usr (현재위치가 /home/ubuntu)
	->상대경로(relative path) 표기법
	:현재 위치부터 도착지까지 적는 방법


clear
: 화면 지우기

cd(change directory)
: 디렉토리 이동
: cd 이동할디렉터리명
: 절대경로(absolute path), 루트경로부터 도착지까지를 적는 표기법
: 상대경로(relative path), 현재경로에서의 상대위치를 적는 표기법
  ..  : 부모 디렉토리
    . : 현재 디렉토리
: 홈 디렉토리로 이동
  -> cd ~, cd


-- vi editor
: 텍스트 에디터
: vi 파일명
입력모드(i : 현재 커서위치, a :커서 다음위치)
명령줄모드(esc)
저장 및 나가기(:w, :q, :wq!)
H(왼쪽)J(아래쪽)K(위쪽)L(오른쪽) 또는 방향키
yy(yank): 한줄 복사
p(paste): 한줄 붙이기
dd(delete): 한줄 삭제
X : 커서 위치의 글자 삭제
명령줄 모드 (esx) :(쌍점) 입력
:w(저장), :q(나가기), :wq(저장하고 나가기)
:wq! (저장하고 강제로 나가기)
검색: /검색어 엔터 -> n 검색어의 다음 위치로 이동
                         -> N 검색어의 이전 위치로 이동

cat
: 파일 내용보기
: cat 파일경로


rm(remove)
: 파일 삭제
: rm 파일명

touch
: 파일 생성
: touch 파일명

cp(copy)
: 파일 복사
: cp 원본 파일경로 대상 파일경로

mv(move)
: 파일 이동 또는 파일명 변경
: mv 원래파일경로 대상파일경로

rm(remove)
: 파일 삭제
: rm 파일경로

rmdir(remove directory)
: 디렉터리 삭제
: rmdir 디렉터리명
  -> 디렉터리 내부에 아무것도 없을 때만 삭제 가능
: rm -rf 디렉터리명
  -> 디렉터리 하위 디렉터리 및 파일 포함
  -> -rf(recursive force)


-- 패키지관리(apt)
sudo apt update: 최신 패키지 저장소 주소로 업데이트
sudo apt install 패키지명: 패키지 설치
-> sudo apt install default-jre ex) 자바 설치
-> sudo(관리자 권한으로 실행)
sudo apt remove 패키지명: 패지키 삭제, 설정파일은 놔 둠
sudo apt purge 패키지명: 패키지 삭제, 설정파일도 삭제
